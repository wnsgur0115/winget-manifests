# 📦 Personal Winget Manifests

This repository contains a collection of personal package manifests for **Winget (Windows Package Manager)**.
It includes useful tools that are either not available in the official repository or require separate management.

## 🚀 App List

### 🛠️ VkDiag
> **Vulkan Layer Diagnostic Tool**
*   **Description**: A tool designed to diagnose Vulkan API layer configurations. It detects faulty layers or driver conflicts and helps resolve them.
*   **Usage**: Essential for troubleshooting game crashes, startup failures, or graphical glitches related to Vulkan.
*   🔗 [Original Repository](https://github.com/13xforever/vkdiag)

### 🗂️ Desktop Fences +
> **Desktop Icon Organizer**
*   **Description**: A utility that organizes desktop icons into grouped containers called "fences," keeping your workspace clean and tidy.
*   **Usage**: Helps categorize scattered icons to improve workflow efficiency. (An open-source alternative to Stardock Fences).
*   🔗 [Original Repository](https://github.com/limbo666/DesktopFences)

# 📦 Personal Winget Manifests

이 저장소는 **Winget(Windows Package Manager)**을 통해 설치할 수 있는 개인 패키지 매니페스트 모음입니다.
공식 저장소에 등록되지 않았거나, 별도의 관리가 필요한 유용한 도구들을 포함하고 있습니다.

## 🚀 App List

현재 등록된 애플리케이션 목록입니다.

### 🛠️ VkDiag
> **Vulkan Layer Diagnostic Tool**
*   **기능**: Vulkan API 레이어 구성을 진단하고, 충돌을 일으키는 잘못된 레이어나 드라이버 문제를 탐지하여 해결을 돕습니다.
*   **활용**: 게임 실행 불가, 그래픽 깨짐 등 Vulkan 관련 오류 발생 시 트러블슈팅용으로 사용됩니다.
*   🔗 [Original Repository](https://github.com/13xforever/vkdiag)

### 🗂️ Desktop Fences +
> **Desktop Icon Organizer**
*   **기능**: 바탕화면의 아이콘들을 '울타리(Fences)'라는 그룹으로 묶어 깔끔하게 정리해 주는 도구입니다.
*   **활용**: 복잡한 바탕화면을 카테고리별로 정돈하여 작업 효율을 높일 때 사용됩니다. (Stardock Fences의 오픈소스 대안)
*   🔗 [Original Repository](https://github.com/limbo666/DesktopFences)
